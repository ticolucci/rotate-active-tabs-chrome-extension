To change the keyboard shortcut, click the button below and follow the instructions.

How to change the shortcut:

  1. Click the "Open Shortcuts Page" button below
  2. A new tab will open with instructions (Chrome doesn't allow direct access to chrome://extensions/shortcuts)
  3. Copy chrome://extensions/shortcuts and paste it in your browser's address bar
  4. Find "Rotate Active Tabs" in the list
  5. Click the pencil icon or shortcut field to set a new shortcut
  6. Return to this page and click "Refresh" to see the updated shortcut
